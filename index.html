HCI Usability & Accessibility Audit

College Website – Analysis & Redesign

Project Overview

This project presents a Human–Computer Interaction (HCI) based usability and accessibility audit of a college website, followed by a focused redesign of the Notices & Updates page to improve clarity, accessibility, and user experience.

Usability Findings (Nielsen’s Heuristics)

  • Lack of clear system feedback for new and important notices.
  • Inconsistent layout, spacing, and typography across pages.
  • High cognitive load due to dense information presentation.
  • Important actions and deadlines are visually buried.
  • No sorting or filtering options for notices.

Accessibility Findings

  • Low color contrast in multiple text sections.
  • Small font sizes affecting readability.
  • Limited keyboard navigation support.
  • Non-semantic structure for screen readers.

Redesigned Homepage (Prototype)

This redesigned homepage layout focuses on reducing cognitive load, improving visual hierarchy, and making key actions immediately visible.

Redesign Rationale

  • Introduce clear visual hierarchy using cards.
  • Highlight important notices using tags and color cues.
  • Improve spacing, typography, and contrast.
  • Reduce cognitive load through simplified layout.
